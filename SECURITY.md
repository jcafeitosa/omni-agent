# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Send details to: `security@omni-agent.dev`

Include:

- Affected component/package
- Reproduction steps
- Impact and expected blast radius
- Proposed mitigation (if any)

We will acknowledge reports within 3 business days.

## Scope

High-priority areas:

- Provider authentication and token handling
- Tool execution and sandbox boundaries
- File/path traversal in tools
- Prompt injection and data exfiltration vectors
